Scroll-N-React

Github // Demo Link

Software: A React Component Library. Its an infinite scroller but with the added benefit of symmetric behavior on both ends of the scrolling component. This library was made in response to other infinite scrollers that continuously fire their fetching function on scroll up. This leads to an assymmetric experience when loading at the bottom vs loading at the top.
